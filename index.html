Les actions possibles :

Pour réaliser l'émojif ci dessous vous pourrez utiliser les méthodes suivantes :

  • Pour remplir une case en noir, vous devez appuyer sur la flèche du haut de votre clavier.
  • Pour remplir une case en jaune, vous devrez cette fois si appuyer sur la flèche du bas de votre clavier.
  • Enfin pour effacer une couleur en cas d'erreur, vous pouvez double-cliquer avec votre souris sur la case de votre choix.

Si vous le souhaitez vous pouvez réaliser d'autres motifs. Enjoy =)
